# COMPOSITECALENDARWORKER

## About

Worker service for the compositecalendar app

## Configuration

Set the following environment variables:

* Runtime Env
  * OAUTH_CLIENT_ID
  * OAUTH_CLIENT_SECRET
  * DB_CONNECTION
  * DB_USERNAME (Optional)
  * DB_PASSWORD (Optional)
* Build Env
  * GOOGLE_CREDENTIALS
  * DOCKERHUB_USERNAME
  * DOCKERHUB_PASSWORD
